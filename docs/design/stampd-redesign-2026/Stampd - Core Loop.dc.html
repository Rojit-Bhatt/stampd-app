Animation spec — for React + motion (spring), all under useReducedMotion
Earn — "the coin lands"

Coin badge springs 0→1.16→1 (stiffness 280, damping 14). +points counts up from 0 over 600ms with a serif odometer feel, then the campaign chip pops (delay 300ms). Balance card fades up last. Reduced-motion: instant, no count-up, no scale.

Redeem — "the exchange"

Distinct on purpose: a rounded voucher tile flips in (rotateY 90→0, spring stiffness 220 damping 18), reward name slides up, remaining balance ticks down to the new figure. Dashed border signals "show this to staff." Reduced-motion: crossfade only.
