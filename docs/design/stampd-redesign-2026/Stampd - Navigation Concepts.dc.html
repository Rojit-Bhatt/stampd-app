Stampd · Navigation — pick one per console

Two tools, two navigations

Today both consoles are the same 250px sidebar. They shouldn't be. The outlet console is a counter tool used one-handed on a tablet/phone under time pressure — its nav should put Earn and Redeem in the thumb zone. The platform console is a dense, data-heavy desk tool — its nav should maximize table width and support fast jumps. Reply with one id per console (e.g. "OB and PA"). No feature is dropped in any concept — only where it lives changes.

Outlet console · counter tool (tablet / phone)
