Green is meaning, not decoration. It marks value and primary action everywhere and is the one color that never changes per tenant. Neutrals carry warmth in dark mode (green-tinted, not pure gray) to keep surfaces from feeling dead.

02

Tenant-color contract

The rule

Tenant hue owns identity — logo tiles, business name, explore banners, a thin card accent bar, the theme of the customer subtree. Green owns value & action — the points figure, the scan button, earn/redeem semantics. A customer never loses "how much can I spend," whatever color the outlet picked.

:root { --tenant: #B8460C; }
--tenant-ink: /* auto: darken(--tenant)
  until contrast ≥ 4.5 on --surface */
--tenant-on: /* white | --ink, whichever
  passes contrast on --tenant */
Survives an ugly color
burnt
indigo
yellow
magenta

A near-white or neon tenant color never lands on text: the algorithm derives --tenant-ink for text roles and picks --tenant-on for fills, so contrast holds. The green value figure is unaffected.
